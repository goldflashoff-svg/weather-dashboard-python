# Weather Dashboard

A Python project that fetches real-time weather information using the Open-Meteo APIs.

## Features

- Search weather by city
- Displays city and country
- Shows temperature
- Shows wind speed
- Handles invalid city names

## Technologies Used

- Python
- requests
- Open-Meteo Geocoding API
- Open-Meteo Weather API

## How to Run

1. Install requests

pip install requests

2. Run the program

python weather_dashboard.py
